# Superstore Sales Performance Dashboard

This repository contains a Power BI sales performance dashboard built from the Sample Superstore dataset.

## Files

- `Superstore Sales Performance Dashboard.pbix` — Power BI report
- `Sample_Superstore.xlsx` — Source dataset
- `Dashboard Screenshot.png` — Preview image

## How to use

1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. If prompted for a data source, point the model to `Sample_Superstore.xlsx`.

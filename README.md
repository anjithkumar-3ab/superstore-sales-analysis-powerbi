# Superstore Sales & Profit Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, product performance, regional distribution, and geographical trends.

The objective of this project is to transform raw retail data into meaningful business insights through data visualization and business intelligence techniques.

## Dashboard Features

### Key Performance Indicators (KPIs)

- Total Sales
- Total Profit

### Sales Analysis

- Monthly Sales Trend
- Sales by Customer Segment
- Sales by City
- Sales by Product

### Regional Analysis

- Quantity Distribution by Region
- Sales Distribution Across States

### Geographic Insights

- State-wise sales visualization using map analytics

## Tools Used

- Power BI Desktop
- Microsoft Excel / CSV Dataset
- Data Visualization Techniques
- Business Intelligence Concepts

## Business Insights

- Consumer segment contributes the highest sales.
- West region records the highest quantity sold.
- New York City generates the highest sales.
- Certain products significantly outperform others in revenue generation.
- Sales exhibit seasonal fluctuations throughout the year.
